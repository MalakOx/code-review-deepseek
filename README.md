# Code Review Assistant (DeepSeek-Coder)

A powerful code review assistant that uses the **DeepSeek-Coder model** via Ollama to analyze code and provide comprehensive feedback, suggestions, and bug fixes.

## Features

- 🔍 **Comprehensive Code Analysis**: Reviews code for bugs, best practices, performance, and security
- 🚀 **FastAPI Backend**: High-performance API with error handling and health checks
- 🎨 **Modern Streamlit Frontend**: User-friendly interface with real-time status monitoring
- 🔧 **Multi-language Support**: Works with Python, JavaScript, Java, C++, Go, Rust, and more
- 📊 **Detailed Feedback**: Provides specific, actionable recommendations
- 📥 **Export Reviews**: Download review reports as Markdown files

## Prerequisites

- Python 3.8+
- [Ollama](https://ollama.ai/) installed and running
- DeepSeek-Coder model pulled via Ollama
